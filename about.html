Про цю роботу

Цей сайт створено як виконання практичної роботи №8: створення сайту за допомогою онлайн-конструктора / CMS. Він містить адаптивний інтерфейс, каталог товарів, простий кошик на JavaScript та інструкцію для публікації.

Використані технології

  • HTML5 + CSS3
  • JavaScript (клієнтський кошик)
  • Адаптивний дизайн (flex / grid)

Інструкція для здачі

  1. Розпакуйте архів і розмістіть файли на GitHub Pages або будь-якому статичному хостингу.
  2. Або завантажте в Wix/Tilda як окремі сторінки або вставте HTML всередину блоків (можливо, знадобиться преміум-аккаунт для повноцінного інтернет-магазину).
  3. Прикріпіть посилання на опублікований сайт та файл APK (якщо ви створите додаток через AppsGeyser) у Moodle.
