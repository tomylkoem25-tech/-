README — Demo Shop (практична робота №8)
---------------------------------------
Що всередині:
- index.html — головна сторінка
- products.html — каталог товарів
- about.html — опис роботи та інструкції
- css/style.css — стилі
- js/app.js — простий кошик (localStorage)
- images/ — SVG-зображення-заповнювачі

Відповідність вимогам практичної роботи №8:
- Тематика: інтернет-магазин / демонстраційний сайт.
- Адаптивність: використано CSS Grid та медіа-запити.
- SEO: мета-теги опису та семантичні теги (main, header, footer).
- Публікація: статичні файли можна розмістити на GitHub Pages або додати в конструктор типу Wix/Tilda (для повноцінного магазину потрібні преміум-опції Wix для прийому платежів).
- Додаток: інструкція в about.html як створити APK через AppsGeyser (див. практичну роботу).

Як здавати в Moodle:
1) Розпакуйте архів і завантажте сайт на публічний хостинг (GitHub Pages або інший).
2) Додайте посилання на опублікований сайт у поле Moodle.
3) (Опційно) За бажання створіть APK через https://appsgeyser.com і додайте файл APK (під час створення вкажіть URL вашого сайту).

Файли в архіві: demo-site.zip
